서비스 소개

Lumen(이하 "앱")은 사용자의 캘린더 데이터를 분석하여 매일 아침 맞춤 하루 브리핑을 제공하는 Android 애플리케이션입니다. 개발자 SIK.(이하 "개발자")은 사용자의 개인정보를 소중히 여기며 본 방침에 따라 처리합니다.

📱 캘린더 데이터는 사용자의 기기 안에서만 처리되며, 외부 서버에 저장되지 않습니다.

수집하는 정보

앱은 아래 정보에 접근합니다.

  • 캘린더 데이터 — 일정 제목, 시작/종료 시간, 참석자 이메일 주소. 분석 목적으로만 사용되며 외부로 전송되지 않습니다.
  • Google 계정 정보 (Google 캘린더 연결 시) — 이름, 이메일, 프로필 사진. 화면에 표시하는 용도로만 사용됩니다.
  • 앱 설정 — 알림 시간, 선택한 캘린더 목록 등. 기기 로컬 저장소(SharedPreferences)에만 저장됩니다.
  • 앱 사용 통계 및 진단 정보 — Google Firebase를 통해 다음 데이터가 수집됩니다. 앱 품질 개선 외 목적으로 사용되지 않으며, 캘린더 일정 제목·회고 본문·할 일 제목 등 식별 가능한 콘텐츠는 절대 포함되지 않습니다.
    • 화면 전환·기능 사용 횟수, 카테고리·버킷 단위의 메타데이터 (Firebase Analytics)
    • 앱 비정상 종료 시 크래시 스택트레이스, 디바이스 모델·OS 버전 (Firebase Crashlytics)
    • 앱 시작 시간·네트워크 응답 속도 등 성능 지표 (Firebase Performance)
    • 푸시 알림 수신용 디바이스 토큰 (Firebase Cloud Messaging)

🚫 광고 식별자(AD_ID)는 수집하지 않으며, 매니페스트에서 명시적으로 차단되어 있습니다.

정보 사용 목적

  • 오늘의 일정 요약 및 집중 시간 분석
  • 관계 인사이트 (오래 연락하지 못한 사람 안내)
  • 이번 주 업무·개인 시간 비율 계산
  • AI 브리핑 생성을 위한 Anthropic API 요청

🤖 AI 브리핑 생성 시 일정 제목과 시간만 Anthropic API로 전송됩니다. 참석자 이메일 등 개인 식별 정보는 전송되지 않습니다.

제3자 제공 및 외부 서비스

  • Anthropic API — AI 브리핑 생성에 사용됩니다. 일정 제목·시간만 전송되며, Anthropic의 개인정보처리방침을 따릅니다.
  • Google Calendar API (Google 캘린더 연결 시) — Google의 개인정보처리방침을 따릅니다.
  • Google Firebase — 앱 품질 개선을 위한 사용 통계·크래시·성능 진단 및 푸시 알림 전달에 사용됩니다. 수집 항목은 위 "수집하는 정보" 섹션을 참고해 주세요. Google의 Firebase 개인정보처리방침을 따릅니다.

앱은 광고 SDK 및 제3자 마케팅 서비스를 포함하지 않습니다.

데이터 보관 및 삭제

  • 캘린더 데이터·앱 설정 등 사용자 콘텐츠는 기기 내 로컬 저장소에만 보관됩니다.
  • 앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.
  • 앱 내 설정 → 로그아웃을 통해 연결을 해제하고 캐시를 즉시 초기화할 수 있습니다.
  • Firebase Analytics 사용 통계는 Google 정책에 따라 최대 14개월, Crashlytics 크래시 로그는 90일 보관됩니다. 해당 데이터에 대한 삭제 요청은 아래 문의 이메일로 보내주시면 처리됩니다.

필요 권한

  • 캘린더 읽기·쓰기 — 일정 분석 및 사용자가 추가한 일정을 캘린더에 반영하기 위해 필요합니다.
  • 알림 — 매일 아침 브리핑·체크리스트 리마인더·루틴 알림 발송에 사용됩니다.
  • 정확한 알람 (SCHEDULE_EXACT_ALARM) — 지정한 시각에 알림이 정확히 울리도록 사용됩니다.
  • 부팅 완료 (RECEIVE_BOOT_COMPLETED) — 기기 재부팅 후 예약된 알림을 다시 등록하기 위해 사용됩니다.
  • 인터넷 — AI 브리핑 생성, Google 로그인, 푸시 알림 수신, Firebase 진단 데이터 전송에 필요합니다.
