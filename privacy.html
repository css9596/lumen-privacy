서비스 소개

Lumen(이하 "앱")은 사용자의 캘린더 데이터를 분석하여 매일 아침 맞춤 하루 브리핑·체크리스트·루틴·회고를 제공하는 Android 애플리케이션입니다. 개발자 SIK.(이하 "개발자")은 사용자의 개인정보를 소중히 여기며 본 방침에 따라 처리합니다.

📱 캘린더 일정·할 일·루틴·회고 본문 등 사용자가 입력·연결한 콘텐츠는 사용자의 기기 안에서만 보관됩니다. AI 응답 생성에 필요한 최소 정보만 외부 API로 전송되며, 앱 안정성·기능 개선을 위한 진단 정보는 Firebase로 자동 전송됩니다(아래 항목 참조).

수집하는 정보

사용자가 직접 제공하거나 연결한 정보

  • 캘린더 데이터 — 일정 제목, 시작/종료 시간, 장소, 참석자 이메일 주소. 화면 표시·분석·AI 응답 생성에 사용됩니다.
  • Google 계정 정보 (Google 캘린더 연결 시) — 이름, 이메일, 프로필 사진. 화면 표시 용도로만 사용됩니다.
  • 사용자 입력 콘텐츠 — 할 일, 매일 루틴, 체크리스트 항목, 저녁 회고 본문. 기기 로컬 저장소(SharedPreferences)에만 보관됩니다.
  • 앱 설정 — 알림 시간, 선택한 캘린더 목록, 테마, 데모 모드 등. 기기 로컬에만 저장됩니다.

자동으로 수집되는 진단·운영 정보

  • 충돌 보고서 (Firebase Crashlytics) — 앱 비정상 종료 시 스택 트레이스, OS 버전, 기기 모델, 앱 버전, 가명 처리된 설치 식별자.
  • 사용 분석 (Firebase Analytics) — 화면 전환, 기능 사용 횟수와 같은 메타데이터. 일정 제목·할 일 텍스트·회고 본문 등 사용자 콘텐츠는 일절 포함되지 않습니다.
  • 푸시 토큰 (Firebase Cloud Messaging) — 앱 재배포 없이 운영 공지·긴급 안내를 보낼 수 있도록 기기별 익명 토큰을 저장합니다.
  • 성능 트레이스 (Firebase Performance) — 앱 시작 시간, 네트워크 응답 시간 등 성능 지표.
  • 원격 설정 (Firebase Remote Config) — AI 장애 시 자동 비활성화 등 운영 플래그를 앱 재배포 없이 적용하기 위한 단방향 설정 수신.

정보 사용 목적

  • 오늘의 일정 요약, 집중 시간 분석, 이번 주 업무·개인 시간 비율 계산
  • 일정 맥락에 맞는 AI 체크리스트·아침 브리핑·저녁 회고 응답 생성
  • 매일 루틴·체크리스트·아침 브리핑 등 로컬 알림 스케줄링
  • 충돌·오류·성능 문제의 원인 파악과 안정성 개선
  • 중요한 운영 공지 발송 및 AI 장애 시 즉시 대응

🤖 AI 응답 생성 시 Anthropic API로 전송되는 정보는 다음과 같습니다:

  • 아침 브리핑·주간 분석: 일정 제목·시간(60일 이내), 최근 회고 톤 요약
  • 체크리스트: 해당 일정의 제목·시간·장소, 과거 같은 유형 일정에서 학습한 패턴 (자주 함께 등장한 사람 이름·반복 방문 장소·이전 체크리스트 항목)
  • 저녁 회고: 사용자가 직접 작성한 회고 본문

Google 계정 이메일·프로필 사진·기기 식별자는 Anthropic으로 전송되지 않습니다. 앱 설정에서 데모 모드를 켜면 모든 AI 호출이 가짜 응답으로 대체되어 외부 전송이 발생하지 않습니다.

제3자 제공 및 외부 서비스

  • Anthropic API — AI 브리핑·체크리스트·회고 응답 생성에 사용됩니다. Anthropic의 개인정보처리방침을 따릅니다.
  • Google Calendar API (Google 캘린더 연결 시) — Google의 개인정보처리방침을 따릅니다.
  • Google Firebase (Crashlytics, Analytics, Cloud Messaging, Performance Monitoring, Remote Config) — 앱 안정성·운영을 위한 진단·메타데이터 전송에 사용됩니다. Firebase의 개인정보 처리 및 Google 개인정보처리방침을 따릅니다.
  • Google Play 서비스 — 앱 내 업데이트 안내(in-app update)에 사용됩니다.

앱은 광고 SDK 및 제3자 마케팅 서비스를 포함하지 않습니다. Firebase Analytics는 광고 식별자(AAID)를 수집하지 않으며, 사용자 콘텐츠를 전송하지 않습니다.

데이터 보관 및 삭제

  • 캘린더·할 일·루틴·체크리스트·회고 등 사용자 콘텐츠는 기기 내 로컬 저장소에만 보관됩니다. 앱을 삭제하면 함께 삭제됩니다.
  • 앱 내 설정 → 로그아웃으로 Google 연결을 해제하고 캐시·콘텐츠를 즉시 초기화할 수 있습니다.
  • Firebase 진단 데이터(Crashlytics·Analytics·Performance·푸시 토큰)는 Google Firebase 서버에 Google이 정한 보존 기간 동안 보관되며, 사용자가 앱을 삭제해도 자동 삭제되지는 않습니다. 해당 데이터의 삭제를 원하시면 아래 문의 이메일로 요청해 주시면 가명 식별자 기준으로 삭제 처리합니다.

필요 권한

  • 캘린더 읽기 (READ_CALENDAR) — 기기 캘린더 일정 분석에 필요합니다.
  • 알림 (POST_NOTIFICATIONS, Android 13+) — 아침 브리핑·체크리스트 리마인더·매일 루틴 알림 발송에 사용됩니다.
  • 정확한 알람 (USE_EXACT_ALARM / SCHEDULE_EXACT_ALARM) — 지정한 시각에 정확히 알림이 울리도록 사용됩니다. 캘린더·리마인더 카테고리 앱에 한해 자동 부여되는 권한입니다.
  • 부팅 완료 (RECEIVE_BOOT_COMPLETED) — 기기 재시작 후에도 예약 알림이 유지되도록 사용됩니다.
  • 인터넷 — AI 응답 생성, Google 로그인, Firebase 통신에 사용됩니다.
