앱에서 직접 삭제하기 권장

Lumen의 모든 데이터는 사용자의 기기에만 저장됩니다. 아래 단계를 따르면 즉시 모든 데이터가 삭제됩니다.

삭제 단계
  1. 앱 실행 후 오른쪽 상단 프로필 아이콘 탭
  2. 설정 화면 진입
  3. 하단 "로그아웃" 탭
  4. 확인 다이얼로그에서 "로그아웃" 선택

로그아웃 즉시 캘린더 연결, 브리핑 캐시, 알림 설정, 저장된 이름 정보가 모두 삭제됩니다.

삭제되는 데이터 및 보관 기간

데이터 유형 삭제 시점 외부 서버 보관
캘린더 분석 캐시 즉시 삭제 없음
Google 로그인 세션 즉시 삭제 없음
알림 설정 즉시 삭제 없음
저장된 연락처 이름 즉시 삭제 없음
선택한 캘린더 목록 즉시 삭제 없음
AI 브리핑 생성에 사용된 일정 정보 Anthropic API 처리 후 보관하지 않음 없음

Lumen은 별도의 서버나 데이터베이스를 운영하지 않습니다. 모든 데이터는 기기 내에서만 처리·저장됩니다.
